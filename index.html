Who We Are

Caldera Labs creates plugins and educational content for web developers to make WordPress more powerful and more fun. We are best known for our popular WordPress plugin Caldera Forms. Team Caldera is led by Josh Pollock and Christie Chirinos.

What We Create

Caldera Forms

Caldera Forms is a drag and drop, responsive form builder for WordPress. Simple and fun for everyone to use with all of the tools a developer needs to do anything with WordPress. Free to use, with a Pro version that reimagines the way WordPress plugins work.

Caldera Learn

Through the eyes of the pros, learn modern WordPress development using the WordPress REST API, object-oriented PHP and JavaScript MVC frameworks from Caldera Labs' lead developer Josh Pollock, Senior WordPress Engineer Roy Sivan, and friends.

Custom Plugin Development

Is the sky the limit? Is space the limit? We make WordPress plugins, that's what we do. Whether you've got an idea for a new plugin, need help improving an existing plugin, want us to customize one of our existing plugins for you, or even want us to review the plugin code you wrote, we're here to help.
