Services

Caldera Labs provides select services for WordPress developers.

Code Reviews

If you're working on a WordPress plugin or theme, it's important to get a fresh set of eyes on the project to avoid accumulating technical debt or introducing security issues. Whether you are looking for guidance to improve your code or need help finishing your first big project, we can help.

All code reviews are carried out by Caldera's lead developer Josh Pollock, a core contributor to WordPress and author of more plugins than he can count.

If you're not a developer and you need help evaluating a vital plugin or theme for your next project, we can help. Before you rely on someone else's code, get an expert opinion on its security, performance and stability.

Get in touch to get started.

Plugin Development

In addition to our plugins, we have also worked on new plugins and new features to plugins by FooPlugins, Postmatic, Pods, Aesop Interactive, WordImpress, GravityView and more.

We also are also available for web and mobile application development using the WordPress REST API, Laravel and AngularJS.
